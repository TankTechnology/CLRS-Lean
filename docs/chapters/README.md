# Supplementary Chapter Notes

Files in this directory are optional maintainer or study notes for selected
chapters.  They are not required for every represented chapter and are not the
source of truth for proof status.

The canonical reader-facing chapter guides are
`CLRSLean/Chapter_XX.lean`.  Current progress comes from
`docs/clrs-proof-progress.csv`, and theorem-level detail belongs in
`docs/proof-map.md`.
